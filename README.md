# Maison Noir

Luxury fine dining restaurant website designed and developed with a modern frontend stack.

🌐 **Live Demo:** https://maison-noir-experience.netlify.app

## Overview

Maison Noir is a premium restaurant website concept inspired by Michelin-star dining experiences. The project focuses on elegant UI design, cinematic motion, responsive layouts, accessibility, and production-ready deployment.
